# Meowdows 95 艺术评论家系统 (AI Art Critic Win95)

> 欢迎体验 **Meowdows 95 艺术评论家系统**！这是一款向经典 Windows 95/98 操作系统、复古像素艺术以及宝可梦卡牌致敬的趣味互动简笔画评估系统。通过对接前沿的大语言模型（LLM）视觉 API，系统会对您的每一幅涂鸦进行幽默调侃、深奥解构或盲目夸赞，并能一键导出精美的卡牌。

---

## 核心特色

### 1. 复古像素画板 (Drawing Board)
- **多功能工具箱**：提供铅笔、画笔、喷枪、橡皮擦工具。
- **油漆桶工具 (Flood Fill)**：内置高性能基于栈的区域填充油漆桶，采用**曼哈顿色彩距离容差匹配算法**（容差设定为 `450`），完美融合线条与填充色的边缘，彻底消除抗锯齿带来的像素空隙。
- **历史记录系统**：支持无限次撤销（Undo/Ctrl+Z）与重做（Redo/Ctrl+Y）操作，方便您进行艺术创作。

### 2. 多样化的 AI 艺术评论家风格 (Critique Styles)
您可以双击桌面的“控制面板”更改 AI 评论家的人格，或选择“随机”让评论家随机切换以下身份：
- **傲慢尖酸 (Arrogant)**：自视甚高的皇家艺术学院院长，满口晦涩学术黑话（解构主义、透视坍塌），冷嘲热讽却文雅有趣。
- **温柔鼓励 (Supportive)**：盲目偏袒的狂热信徒，无论画得多糟都用歇斯底里、热泪盈眶的传销式造神语言夸赞。
- **诗意浪漫 (Poetic)**：随时会崩溃的先锋现代诗人，用破碎的宏大意象将简笔画解读为对生命流逝的哀歌。
- **深奥哲学 (Philosophical)**：深陷虚无主义危机的哲学家，强行使用尼采、黑格尔理论及西西弗斯的荒诞隐喻解构画作。
- **无厘头搞笑 (Nonsense)**：精神状态极度不稳定的网络乐子人，逻辑断裂，用跳跃的跨物种比喻爆笑吐槽。

### 3. 宝可梦式画作珍藏卡 (Pokemon-Style Collectible Cards)
在“画廊陈列室.exe”中，您可以为心仪的画作生成并下载精美的复古卡牌：
- **四种稀有度划分**：根据 AI 评分动态赋予卡牌主题边框与闪卡光效：
  - **SSR (90+分)**：`✦ 传世纯金闪卡 ✦`，尊贵金黄渐变及全息粒子闪烁效果。
  - **SR (80+分)**：`✦ 传奇全息闪卡 ✦`，优雅香槟金渐变效果。
  - **R (50+分)**：`✦ 银色珍藏版 ✦`，经典冰蓝渐变框。
  - **C (50分以下)**：`✦ 无限普卡 ✦`，复古白银风格。
- **动态物理属性计算**：
  - 针对内置的数十种词语，预设了搞笑的官方卡牌身高与体重属性（例如：`西瓜` 重 4.5kg，`太阳` 重 1.98e30kg，`吸血鬼` 高 1.8m）。
  - 对于其他词语，系统会通过字符哈希（Hash）算法**自动计算出稳定唯一的专属身高、体重和图鉴编号**，确保趣味十足的同时数据前后一致。
  - 支持将 AI 生成的自定义词库同步携带身高、体重设定。

### 4. 拟真 Windows 95 桌面环境 (Desktop OS Environment)
- **多窗口管理**：支持多窗口并存、最小化、最大化、拖拽移动以及活动窗口置顶激活。
- **实时同步的画廊**：在绘图结果页一键“保存到画廊”后，已打开的“画廊陈列室.exe”窗口会基于状态触发器**实时局部刷新**，无需重开即可查阅新画作。
- **全局通知**：所有的保存配置、保存画作及生成卡片行为全部改用非阻塞式 **Windows 95 风格 Toast 浮窗提示**，保障交互流畅度。

## 技术栈

- **核心框架**：[React](https://react.dev/) + [Vite](https://vite.dev/)
- **类型安全**：TypeScript
- **样式系统**：Tailwind CSS (经典 Win95 复古像素主题样式)
- **数据持久化**：IndexedDB (采用原生 IndexedDB 进行画作与系统配置文件的本地离线保存)
- **图形绘制**：HTML5 Canvas API
- **图标包**：[Iconify](https://iconify.design/) (@iconify/react)

## 快速上手

### 1. 安装依赖
请确保您本地已安装 [Node.js](https://nodejs.org/) 环境。在项目根目录下运行：
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```
打开浏览器访问控制台输出的地址即可（通常是 `http://localhost:5173`）。

### 3. 构建生产包
```bash
npm run build
```
编译产物将输出至 `dist` 目录中。

## 评论家 API 配置指南

系统支持自定义 API 节点，如果您使用的是国外大语言模型（如 Gemini、GPT 或 Claude），请按如下操作：
1. 双击桌面上的 **「控制面板.lnk」** 窗口。
2. 选择接口提供商：
   - **OpenRouter**：默认推荐，支持一键拉取数百个低成本或免费的优秀模型。
   - **Custom**：支持任何兼容 OpenAI 格式的第三方 API 代理端点（如 OneAPI 等）。
3. 填入您的 **API 密钥 (API Key)** 及 **API 地址 (API URL)**。
4. 点击 **“拉取模型列表”** 获取最新模型，保存配置即可开始创作。
